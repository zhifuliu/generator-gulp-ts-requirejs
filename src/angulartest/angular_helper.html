下面是 angular 菜鸟教程中用到的 angularjs 指令: ng-app ng-bind: 绑定 html 元素到应用程序数据 ng-bing-html: 绑定 html 元素的 innerHTML 到应用程序数据，并移除 html 字符串中危险字符 ng-bind-template: 规定使用模板替换的文本内容 ng-blur: blur 事件的行为 ng-change: 在内容发生变化时要执行的表达式 ng-checked: 规定元素是否被选中 ng-clss: 指定 html 元素使用的 css 类 ng-class-even: 类似于 ng-class，但只在偶数行起作用 ng-class-odd: ng-click: ng-cloak: 在应用正要加载时防止闪烁 ng-controller: ng-copy: 规定拷贝事件的行为 ng-csp: 修改内容安全策略 ng-cut: 规定剪切事件的行为 ng-dbclick: ng-disabled: ng-focus: ng-form: 指定 HTML 表单继承控制器表单 ng-hide: ng-href: 为 a 元素指定链接 ng-if: 如果条件为 false 移除 html 元素 ng-include: ng-init: ng-jq: 定义应用必须使用到的库，如 jquery ng-keydown: 规定按下键事件的行为 ng-keypress: ng-keyup: ng-list: 将文本转换为列表 ng-model: 绑定 HTML 控制器的值到应用数据 ng-model-options: 规定如何更新模型 ng-mousedown: ng-mouseenter: ng-mouseleave: ng-mousemove: ng-mouseup: ng-non-bindable: 规定元素或元素不能绑定数据 ng-open: 指定元素的 open 属性 ng-options: 在 select 列表中指定 options ng-paste: 规定粘贴事件的行为 ng-pluralize: 根据本地化规则显示信息 ng-readonly: ng-repeat: ng-selected ng-show: ng-src: img 的 src 属性 ng-srcset: img 的 srcset 属性 ng-style: ng-submit: 规定 onsubmit 事件发生时执行的表达式 ng-switch: 规定显示或隐藏子元素的条件 ng-transclude: 规定填充的目标位置 ng-value: 规定 input 的值 AngularJS 事件：键盘事件、鼠标事件 AngularJS 验证属性: $dirty $invalid $error AngularJS 全局 API：转换 比较 json 基础

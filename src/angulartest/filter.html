过滤器可以使用一个管道字符(|)添加到表达式和指令中
            angularJS 过滤器可用于转换数据：currency / filter / lowercase / orderBy / uppercase


表达式中使用过滤器

原始姓名：{{name}}
大写姓名: {{name | uppercase}}
小写姓名: {{name | lowercase}}

向指令添加过滤器：
个人信息原始顺序:

  • {{x.name + ' ; ' + x.age}}
个人信息按年龄排序:
  • {{x.name + ' ; ' + x.age}}
个人信息按年龄排序(并将名字转大写,要在循环里面添加过滤器啊！！):
  • {{(x.name | uppercase) + ' ; ' + x.age}}

过滤输入:

  • {{x.name + ' ; ' + x.age}}
